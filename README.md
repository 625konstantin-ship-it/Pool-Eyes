# Учёт параметров бассейна

Простой сайт для контроля pH, хлора, температуры и проблем с водой. Работает в браузере, данные сохраняются локально.

## Как открыть

### Способ 1 (обязательно для входа и почты)
1. Дважды кликните **`запуск.command`**
2. Не закрывайте чёрное окно терминала
3. Откройте в браузере: **http://localhost:8080**

> Не открывайте `index.html` двойным кликом — вход может не работать, потому что данные хранятся в другом месте.

## Вход и сброс пароля

- При **регистрации** укажите email — он нужен для восстановления пароля
- **Забыли пароль?** → вкладка «Сброс пароля» → код на email
- Для отправки писем скопируйте `data/smtp.json.example` → `data/smtp.json` и укажите SMTP (Gmail, Yandex и т.д.)
- Если почта не настроена, код показывается на экране (для теста)

## Возможности

- **Регистрация и вход** — у каждого пользователя свои бассейны
- Несколько бассейнов с именами и объёмами
- Измерения: pH, свободный хлор, температура
- **История химии** — что добавили, сколько и комментарий (замена фильтра и т.д.)
- Рекомендации по параметрам и проблемам воды
- Графики и история измерений

## Нормы (ориентир)

- pH: 7.2–7.6
- Свободный хлор: 1.0–3.0 мг/л
- Температура: 24–28 °C

Рекомендации носят справочный характер — сверяйтесь с инструкциями на химии.
