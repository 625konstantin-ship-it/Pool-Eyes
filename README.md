# Учёт параметров бассейна

Сайт для контроля pH, хлора, температуры и проблем с водой. Данные хранятся в **Supabase** — после входа доступны с любого устройства.

## Настройка Supabase (один раз)

1. Создайте проект на [supabase.com](https://supabase.com)
2. **SQL Editor** → выполните скрипт из `supabase/schema.sql`
3. **Authentication → Providers → Email** — включите вход по email
4. **Authentication → URL Configuration** — добавьте URL вашего сайта в **Redirect URLs** (для сброса пароля)
5. **Project Settings → API** — скопируйте URL и anon key в `config.js`:

```js
const SUPABASE_URL = 'https://xxxxx.supabase.co';
const SUPABASE_ANON_KEY = 'eyJ...';
```

> Можно скопировать `config.example.js` → `config.js`, если файла ещё нет.

## Как открыть

### Локально
1. Дважды кликните **`запуск.command`**
2. Откройте **http://localhost:8080**

### Онлайн
Загрузите файлы на хостинг (GitHub Pages, Netlify и т.д.) и укажите URL сайта в Supabase.

## Вход и регистрация

- **Регистрация** — email и пароль (минимум 6 символов)
- **Вход** — тот же email и пароль
- **Сброс пароля** — ссылка приходит на email (настройте SMTP в Supabase → Authentication → Email)

## Возможности

- Несколько бассейнов с именами, объёмом и типом обработки (хлор / перекись)
- Измерения pH, хлора/перекиси, температуры с графиками
- История химии с комментариями
- Проблемы воды и рекомендации
- Карта и маршрут до бассейна

## Файлы

| Файл | Назначение |
|------|------------|
| `index.html` | Страница |
| `app.js` | Логика приложения |
| `supabase-db.js` | Работа с Supabase |
| `config.js` | Ключи Supabase (не публикуйте в открытый репозиторий) |
| `supabase/schema.sql` | Таблицы и политики безопасности |
| `запуск.command` | Локальный сервер |

## Нормы (ориентир)

- pH: 7.2–7.6
- Свободный хлор: 1.0–3.0 мг/л
- Перекись: 30–75 мг/л
- Температура: 24–28 °C

Рекомендации носят справочный характер — сверяйтесь с инструкциями на химии.
