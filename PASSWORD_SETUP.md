# Смена пароля — настройка Supabase

## Как это работает для пользователя

1. Войти на сайт → иконка пользователя → **Сменить пароль**
2. Подтвердить отправку → письмо приходит на email
3. Открыть письмо → **перейти по ссылке**
4. На сайте откроется форма **«Новый пароль»**
5. Ввести пароль дважды → **Сохранить пароль**

---

## Шаг 1. Redirect URLs в Supabase (обязательно)

1. [supabase.com](https://supabase.com) → ваш проект  
2. **Authentication** → **URL Configuration**  
3. **Site URL** — основной адрес сайта, например:  
   `https://625konstantin-ship-it.github.io/Pool-Eyes/`  
4. **Redirect URLs** — добавьте **оба** адреса:

```
http://localhost:8080/
https://625konstantin-ship-it.github.io/Pool-Eyes/
```

5. **Save**

> Без этих URL ссылка из письма не откроет форму смены пароля.

---

## Шаг 2. Email в Supabase

1. **Authentication** → **Providers** → **Email** — включён  
2. **Authentication** → **Email Templates** → **Reset Password**  
   - можно оставить шаблон по умолчанию  
   - в письме должна быть ссылка `{{ .ConfirmationURL }}`

На бесплатном тарифе письма идут через Supabase (лимит ~2–4 в час).  
Для частой смены пароля настройте свой SMTP: **Project Settings → Authentication → SMTP Settings**.

---

## Шаг 3. Обновите файлы на сайте

Загрузите на GitHub (или проверьте локально через `запуск.command`):

- `index.html`
- `app.js`
- `supabase-db.js`

---

## Проверка локально

1. Запустите **`запуск.command`**
2. Откройте **http://localhost:8080/**
3. Войдите → иконка → **Сменить пароль**
4. Проверьте почту → ссылка должна вести на `http://localhost:8080/#...type=recovery`
5. Должна открыться форма **«Новый пароль»**

---

## Если не работает

| Проблема | Решение |
|----------|---------|
| Письмо не приходит | Проверьте спам; подождите; лимит Supabase |
| Ссылка открывает сайт, но нет формы пароля | Добавьте URL в **Redirect URLs** (шаг 1) |
| «Слишком много попыток» | Подождите 1–2 часа |
| Ссылка ведёт на GitHub, а вы тестируете локально | Используйте локальный `запуск.command` или добавьте оба URL |
